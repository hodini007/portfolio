# OrbiSense (Satellite Tracking Application)

- Python, Skyfield, CustomTkinter, Matplotlib

Developed an interactive satellite tracking suite using the Skyfield SGP4 physics engine to predict passes and visualize ground-tracks for Weather satellites, ISS, Starlink, and Amateur Radio. Modular design with TLE caching (24-hour TTL), root-finding for precise horizon-crossing calculations, and a threaded CustomTkinter dark-mode UI with embedded Matplotlib 2D maps. Originally built to intercept weather-satellite telemetry during severe cyclones.
