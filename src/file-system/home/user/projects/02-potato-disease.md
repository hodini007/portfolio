# Potato Disease Detection System

- Python, CNN/Deep Learning, Streamlit, Tensorflow/Keras

Built an AI-powered Streamlit web application that classifies potato leaf diseases (Early Blight, Late Blight, Healthy) using a custom CNN trained on the PlantVillage dataset. Features 256x256 RGB inputs with built-in normalization, Conv2D and MaxPooling2D layers, and dual-input modes (file upload and live camera feed). Architecture is extensible to 10+ plant species.
